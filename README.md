# KimiOS Linux Desktop

A pixel-perfect Linux desktop environment web app inspired by the Kimi Agent interface.
Built with React (UMD) — **zero build step required**, open `index.html` directly.

## Features
- GNOME-style desktop with animated abstract wallpaper
- Draggable windows with macOS traffic-light controls (close / minimize / maximize)
- 7 working apps: Terminal, Files, Text Editor, Web Browser, Settings, Calendar, Trash
- Functional terminal with 20+ bash commands (`ls`, `cd`, `cat`, `neofetch`, `git`, `history`…)
- Activities overlay with search
- Bottom dock with Kimi Agent / Replay / Remix branding
- Live clock and system tray icons

## Quick Start
```bash
git clone https://github.com/rintuchowdory/kimios-linux-desktop
cd kimios-linux-desktop
# Just open index.html in a browser — no npm install needed
```

## Deploy to GitHub Pages
Push repo → Settings → Pages → Branch: main / root → Save.
Live at: `https://rintuchowdory.github.io/kimios-linux-desktop`

## Tech Stack
- React 18 + ReactDOM 18 (CDN via cdnjs)
- Pure inline CSS / no Tailwind
- No build step, no dependencies
